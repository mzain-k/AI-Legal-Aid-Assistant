# QalamAI 🖊️
### AI-Powered Study Companion for FBISE 10th Grade Students

QalamAI (قلم) is an intelligent study assistant built for Pakistani Matric students 
preparing for FBISE SSC Part 2 examinations. Students can ask conceptual questions, 
take quizzes from past papers, get instant feedback on wrong answers, and receive 
a personalized weak topic summary — all in Urdu and English.

---

## Features
- 📚 Ask questions from your actual FBISE textbooks (RAG-powered)
- 📝 Take quizzes generated from real past papers
- ✅ Get instant explanations for wrong answers
- 📊 Receive a weak topic summary after every quiz
- 🇵🇰 Full Urdu and English support

---

## Subjects Covered
FBISE SSC Part 2 (10th Grade)
- Physics, Chemistry, Biology
- Mathematics, Computer Science
- English, Urdu, Pakistan Studies, Islamiat

---

## Tech Stack
- Python 3.11
- Google Gemini API
- LangChain + ChromaDB (RAG pipeline)
- CrewAI (Multi-Agent System)
- Gradio (Web Interface)
